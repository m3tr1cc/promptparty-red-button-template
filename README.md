# PromptParty Red Button Template

A polished standalone Next.js template featuring a centered 2.5D red button interaction on a pure black canvas.

## Stack

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- Framer Motion

## Local Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Scripts

```bash
npm run dev
npm run lint
npm run build
```

## Project Structure

- `src/app/page.tsx` - single-page PromptParty interaction
- `src/components/pressable-button.tsx` - reusable animated button component
- `src/app/globals.css` - global canvas and base styling

## Notes

This template intentionally has no backend, database, authentication, payments, or server actions.
